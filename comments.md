# Chú thích

Single-line comment:

```python
# This is single-line comment
x = 10
```

Inline comment:

```python
x = 10  # 10 is being assigned to x
```

Commenting out

Trước:

```python
number_of_people = 10
```

Sau:

```python
# number_of_people = 10
```

Chú thích nhiều dòng:

```python
'''This is a
multiline comment'''

# hoặc

"""This is also a
multiline comment"""
```
